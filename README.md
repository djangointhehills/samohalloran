# Sam O'Halloran — Jazz Guitarist

Personal website for Melbourne jazz guitarist Sam O'Halloran. Static single-page site built with HTML, CSS, and vanilla JavaScript, hosted on GitHub Pages.

## Structure

```
├── index.html
├── css/styles.css
├── js/script.js
├── assets/images/
├── CNAME
└── .gitignore
```

## Local Development

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

Pushes to `main` deploy automatically via GitHub Pages. Custom domain configured via `CNAME` → `samohalloran.com`.
